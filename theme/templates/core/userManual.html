Phase 1 Data Setup & Management

Before generating schedules, ensuring your core data is accurate is critical. The AI semantic matching and Google OR - CPSAT scheduler relies on this foundational information.

1.1 Instructors

Create faculty profiles, assign ranks, set employment status and define permanent employees workload limits.

1.2 Curriculum & Subjects

Define study plans and add subjects with their units and other requirements.

Curriculums Subjects

1.3 Semesters

Create and select an active academic period (e.g., 1st Sem 2025-2026).

Manage Semesters →

Phase 2 Pre-Scheduling & Constraints

Set the specific requirements for the current semester, including external blocks and section counts.

2.1 GenEd Schedules

Input fixed schedules for subjects handled by other departments. These act as "Hard Constraints".

Manage GenEd →

2.2 Subject Offerings

Tune the priority of the AI semantic matcher to prioritize certain criteria and set the instructors overload limits.

  2. 2
